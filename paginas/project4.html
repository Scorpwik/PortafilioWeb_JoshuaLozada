CONSERVACIÓN GALÁPAGOS

Este proyecto de animación digital fue desarrollado para una campaña de conservación natural en las Islas Galápagos. Participé en un concurso oficial donde mis 3 clips fueron seleccionados como ganadores. El mayor reto y logro de esta pieza fue adaptar la comunicación visual para nuestro cliente final: los niños de las escuelas locales, creando un lenguaje visual educativo, dinámico y muy atractivo.

Rol

Animador / Motion Grapher

Público

Infantil / Educativo

Software Usado:

After Effects Illustrator

Técnicas Implementadas:

  • Animadores 3D: Uso de herramientas nativas para dar volumen a los elementos.
  • Curvas de Velocidad: Ajuste en el editor de gráficos para lograr animaciones orgánicas y fluidas.
  • Ilustración Vectorial: Creación de assets amigables para niños.
